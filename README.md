### <h1>Longread</h1>
Interactive longread using HTML, CSS (SASS), JS (Swiper, wow, animate) and BEM methodology for naming.

### Install the dependencies:
```
$ npm install
```
### Start a website:
```
$ npm run dev
```
### Make a build
```
$ npm run build
```

### Deployment on Vercel
